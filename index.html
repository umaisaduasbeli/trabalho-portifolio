Quem sou eu?

Oiê! Meu nome é Isabelly, uma pessoa que acredita que nossas histórias não cabem em uma só área. Tenho formação em Educação Física pela UFSCar, onde aprendi a valorizar o movimento, a convivência e o trabalho em equipe, essas são as lições que carrego comigo em tudo que faço.

Minha curiosidade por tecnologia começou ainda no ensino médio, quando tive meus primeiros contatos com programação, mexendo com Arduino e participando da OBR (Olimpíada Brasileira de Robótica). Sou apaixonada por tecnologia e por tudo que envolve criar, aprender e inovar.

    Projeto - Locadora de Veículos

    Esse foi um projeto realizado durante a disciplina de Banco de Dados, onde realizei a implantação de um banco relacional, desenvolvi a modelagem de dados (modelo conceitual), criei tabelas, apliquei consultas SQL e testei operações de inserção, atualização e exclusão, garantindo a integridade e a consistência das informações.

    Acesse em meu GitHub.

    Projeto - Lógica de Programação

    Projeto acadêmico criado para uma atividade prática da disciplina de Lógica de Programação e Algoritmos, do meu curso de Análise e Desenvolvimento de Sistemas. Nele resolvo problemas simples usando Python, aplicando entradas de dados, condicionais, laços de repetição e funções. O objetivo foi fortalecer a base de pensamento lógico e versionamento de código no GitHub.

    Acesse em meu GitHub.

    Projeto - Primeiros testes (Git/GitHub)

    Repositório criado para praticar os primeiros comandos de Git e aprender a usar o GitHub na prática. Além dos testes de versionamento, commits, branches e push, também inclui um botão extremamente simples, feito com HTML, CSS e JavaScript, para exercitar a integração entre as linguagens e o uso de interatividade básica na página.

    Acesse em meu GitHub.
